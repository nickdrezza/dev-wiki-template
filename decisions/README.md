# Decisions (ADRs)

Architecture Decision Records — short docs capturing **why** we chose what we chose. Read
these to understand the reasoning behind a past decision.

ADRs are **immutable**: don't rewrite one to reflect a new reality. If a decision changes,
write a new ADR that supersedes the old one (and link them). Number files sequentially
(`NNNN-slug.md`).

# Notebooks

Personal scratch space — one folder per person. **No template, no review bar, no frontmatter
required.** Use it for working notes, drafts, and thinking-out-loud that isn't ready (or may
never be ready) to become a real wiki doc.

Convention: create a folder named for yourself (e.g. `notebooks/your-name/`) and keep your
notes there. When something in your notebook matures into durable team knowledge, promote it
into the right folder ([`guides/`](../guides/), [`systems/`](../systems/), etc.) following
the normal conventions.
